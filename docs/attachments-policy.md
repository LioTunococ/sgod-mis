# Attachments Policy

Allowed file types:
- PDF, DOCX, XLSX, JPG/PNG

Max file size: 10 MB per file (can change later)

UI:
- Drag & drop or click to upload
- Show list of uploaded files with remove option
- Store in /media/submissions/<school>/<submission_id>/

Security:
- Sanitize filename
- Virus-scan hook (optional future)
