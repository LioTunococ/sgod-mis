# UX Copy (Phase 1)

Buttons:
- Save Draft
- Submit
- Continue
- View
- Note (Section only)
- Return with remarks (Section only)

Badges:
- Draft (gray)
- Submitted (blue)
- Returned (orange)
- Noted (green)

Helper text:
- Returned: 'This report was returned by the Section. Please review the remarks and resubmit.'
- Noted: 'This report has been noted by the Section.'

## Demo Walkthrough Script

1. Sign in as demo_school_head (School Head) and open the seeded SMME submission from the dashboard.
2. Click through the Projects tab to highlight pre-filled data, then submit the report.
3. Switch to demo_section_admin (Section Admin) and review the submission via the review queue.
4. Demonstrate returning the submission with remarks, then resubmit as the School Head to show the loop.
5. Back in the Section Admin role, note the updated submission and show the status timeline.
6. Jump to the PSDS/SGOD dashboards to surface the same record, ending with the export action for evidence.
